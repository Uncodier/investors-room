```markdown
    <div className="nx-flex nx-justify-center nx-space-x-8">
      <div className="nx-text-center">
        <div className="nx-text-2xl nx-font-semibold nx-text-green-600">+{QOQ_GROWTH}%</div>
        <div className="nx-text-sm nx-text-gray-600">QoQ Growth</div>
      </div>
      <div className="nx-text-center">
        <div className="nx-text-2xl nx-font-semibold nx-text-green-600">+{YOY_GROWTH}%</div>
        <div className="nx-text-sm nx-text-gray-600">YoY Growth</div>
      </div>
    </div>
  </div>
</div>

### Key Metrics Breakdown

| Metric | {CURRENT_QUARTER} | {PREVIOUS_QUARTER} | Change |
|--------|---------|---------|---------|
| **Total ARR** | ${NEW_ARR} | ${PREVIOUS_ARR} | +{QOQ_GROWTH}% |
| **New ARR** | ${NEW_ARR_AMOUNT} | ${PREVIOUS_NEW_ARR} | +{NEW_ARR_GROWTH}% |
| **Expansion ARR** | ${EXPANSION_ARR} | ${PREVIOUS_EXPANSION} | +{EXPANSION_GROWTH}% |
| **Churned ARR** | -${CHURNED_ARR} | -${PREVIOUS_CHURNED} | {CHURN_CHANGE} |
```

**Placeholder Values to Replace:**
- `{NEW_ARR}` - Current ARR (e.g., $3.2M)
- `{QUARTER}` - Current quarter (e.g., Q1)
- `{YEAR}` - Current year (e.g., 2025)
- `{QOQ_GROWTH}` - Quarter-over-quarter growth percentage
- `{YOY_GROWTH}` - Year-over-year growth percentage
- `{CURRENT_QUARTER}` - Current quarter label
- `{PREVIOUS_QUARTER}` - Previous quarter label
- etc.

### Recommended Tools for Updates

#### Google Sheets Integration
- Set up formulas for QoQ/YoY calculations
- Use Apps Script to generate markdown
- Link with your CRM/analytics tools

#### Zapier Workflows
- Auto-pull metrics from Stripe, Salesforce, etc.
- Generate template updates automatically
- Send reminders for quarterly updates

#### GitHub Actions
- Auto-deploy updates to your investor room
- Version control for metric changes
- Automated backups of previous quarters
